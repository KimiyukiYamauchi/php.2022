# Q2
0以上23以下の整数を受け取り以下のメッセージを表示するプログラム(q2.php)

- 22、23、0以上6未満 => おやすみなさい
- 6以上11未満 => おはようございます
- 11以上16未満 => こんにちは
- 16以上22未満 => こんばんは

``` php
<?php

$input;
do {
  echo '0以上23以下の整数を入力して下さい => ';
  $input = trim(fgets(STDIN));
} while (!($input >= 0 and $input <= 23));

echo $input;

```
