## 条件分岐

- if文を使った分岐

``` php
<?php

// $score = 85;
// $score = 70;
$score = 40;

if ($score >= 80) {
  echo 'Great!' . PHP_EOL;
} elseif ($score >= 60) {
  echo 'Good!' . PHP_EOL;
} else {
  echo 'OK!' . PHP_EOL;
}
```
